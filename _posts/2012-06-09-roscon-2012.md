I started learning [ROS](http://www.ros.org/wiki/About ros.org) (the Robot Operating System) some time ago and I have to say it is an awesome system. It makes it very easy to reuse code for robotic projects and to benefit from tons of work done in robotics so far.

{% include fig.html img="carlitoscontraptions.com" caption="ROSCon 2012" id=page.id %}

When I saw that Willow Garage (and now the OSRF) organized [ROSCon](http://roscon.ros.org/), a ROS developer conference, I couldn't help but assist. Needless to say, it was a blast, I had the chance to learn a lot, and got a bunch of robotic swag.
