A collection of git tricks I keep looking up.

## Tags

```bash
git tag tagname && git push --tags              # create and push
git tag --delete tagname                        # delete local
git push --delete origin tagname                # delete remote
```

One-liner to replace a tag:

```bash
tagname=v7.2.0 && git tag --delete "$tagname" && \
  git push --delete origin "$tagname" && \
  git tag "$tagname" && git push --tags
```

## Stash Selected Files

```bash
git stash -- filename.ext another_filename.ext
```

## Delete a Submodule

```bash
git submodule deinit $submodulename
git rm $submodulename  # add --cached if already removed from disk
git commit -am "Remove submodule"
```

## Prune Stale Remote Branches

```bash
git remote prune origin
```

## Squash a Merge Commit

If the last commit is a merge and you want to squash it:

```bash
git reset --hard HEAD~1
git cherry-pick -m 1 HEAD@{1}
```

## Restore Files by Pattern

Reset all files matching a pattern to the version on `main`:

```bash
find . -name '*.html' -exec git restore --source=main -- {} +
```
